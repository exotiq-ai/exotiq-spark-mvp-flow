## 📝 **Technical Notes for Designer**

### **SVG Best Practices:**
1. Use single path when possible (easier to manipulate)
2. Remove all unnecessary groups/layers
3. Set viewBox to "0 0 100 100" (or appropriate ratio)
4. Optimize with SVGO or similar tool
5. Ensure it scales perfectly at any size

### **PNG Best Practices:**
1. Export at exact pixel dimensions (no scaling)
2. Use transparent background (alpha channel)
3. Optimize with TinyPNG or ImageOptim
4. Test on both light and dark backgrounds
5. Ensure crisp edges (no anti-aliasing artifacts)

### **Color Accuracy:**
- Use exact hex values provided
- Test on multiple screens/devices
- Ensure Gulf Blue matches brand guidelines
- Performance Orange should be vibrant, not muted
