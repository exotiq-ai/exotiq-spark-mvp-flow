## 🚀 QUICK START GUIDES

### **Setup MCP (10 minutes):**

1. Read: `MCP_BREAKTHROUGH_OPUS_COMPLETE.md` (Section 5)
2. Get Supabase Personal Access Token
3. Configure in ElevenLabs:
   - Type: **Streamable HTTP**

   - Header: `Authorization: Bearer YOUR_TOKEN`
