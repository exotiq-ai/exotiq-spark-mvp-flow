# ⚡ QUICK START - EXCELLENCE FEATURES

## 🚀 1-MINUTE SETUP

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Open browser
# http://localhost:8080
```

---

## 🎯 TRY THESE FEATURES

### **Command Palette (Cmd+K)**
```
1. Press Cmd+K (or Ctrl+K)
2. Type "new booking"
3. Press Enter
```
**Result:** Instant navigation, no mouse needed!

### **Onboarding Flow**
```javascript
// In browser console:
localStorage.removeItem('exotiq-onboarding-completed');
location.reload();
```
**Result:** 3-step tour with confetti celebration!

### **Progressive Disclosure**
```
1. Scroll to Revenue Widget
2. Click "Show More"
3. See tachometer, top performers, trends
```
**Result:** Detailed insights without clutter!

### **Automotive Design**
```
1. Look at dashboard banner
2. See racing stripe at bottom (Gulf Blue → Orange)
3. Check utilization tachometer in Revenue Widget
```
**Result:** Luxury automotive feel!
